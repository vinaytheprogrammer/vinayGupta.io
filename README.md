
# Portfolio - Vinay Gupta


## Instructions

### Setup

```shell
git clone https://github.com/vinaytheprogrammer/vinayGupta.git
cd cleanfolio
```

```shell
npm install
npm start
```


To install and launch the project, run these commands:

```shell
yarn
yarn start
```
